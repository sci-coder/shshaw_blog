# Quarto Content Directory

Place your Quarto-rendered HTML files here. They will be accessible at:
`https://sci-coder.github.io/shshaw_blog/quarto-example/`

## How to add Quarto content:

1. Create your Quarto document (.qmd file)
2. Render it to HTML: `quarto render your-document.qmd`
3. Copy the output HTML and resources to this directory
4. Link to it from your Hugo posts or pages

Example structure:
```
static/
  quarto-example/
    analysis.html
    analysis_files/
      figure-html/
      libs/
```
